How it works

Knowledge that links itself.

Write naturally. Aveline turns @mentions into edges, so context compounds instead of scattering across tabs and threads.

Linked by @mention

Reference oncall or auth-flow and the link is made. The graph is the source of truth, not a side effect.

A clean read

Humans get a calm, fast wiki. No markup soup — just pages that are pleasant to skim, search, and trust.

An API for agents

Agents traverse the same graph over a typed API — fetching staging-creds or a postmortem with full context attached.
